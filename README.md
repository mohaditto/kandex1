# Kandex

Plataforma web orientada a la organización de tareas bajo el marco de trabajo Kanban, diseñada para centralizar actividades académicas y personales.

## Stack Tecnológico

- Backend: Node.js con Express.js
- Frontend: EJS con Bootstrap 5
- Base de Datos: MySQL
- Arquitectura: MVC

## Instalación

1. Clona el repositorio.
2. Ejecuta `npm install` para instalar dependencias.
3. Configura las variables de entorno en `.env`.
4. Crea la base de datos ejecutando `schema.sql` en MySQL.
5. Ejecuta `npm start` para iniciar el servidor.

## Uso

- Regístrate o inicia sesión.
- Crea equipos y asigna tareas.
- Usa el tablero Kanban para gestionar tareas con drag & drop.
